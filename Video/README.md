# Project Demonstration Video

This folder contains the Tinkercad simulation demonstrating the working of the Automatic Plant Watering System.

## Demonstration Includes

- System startup
- Soil moisture sensing
- LCD display operation
- Automatic water pump activation
- LED status indication
- Buzzer alert
- Automatic pump shutdown when sufficient moisture is detected

> **Note:** This project was developed and tested using the Tinkercad simulation environment.

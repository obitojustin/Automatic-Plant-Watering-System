# 🌱 Automatic Plant Watering System

An IoT-based Automatic Plant Watering System that monitors soil moisture levels and automatically controls a water pump to irrigate plants, reducing water wastage and enabling smart irrigation.

---

## 📌 Project Overview

The Automatic Plant Watering System is designed to automate plant irrigation using a soil moisture sensor and a microcontroller. The system continuously measures soil moisture and activates a water pump whenever the soil becomes dry. Once the desired moisture level is reached, the pump is automatically turned off.

This project helps conserve water, reduces manual effort, and ensures healthy plant growth.

## ⚙️ Working Principle

1. The soil moisture sensor measures the moisture level.
2. The microcontroller reads the sensor value.
3. If the soil is dry, the relay turns ON the water pump.
4. Water is supplied to the plant.
5. Once sufficient moisture is detected, the pump automatically turns OFF.
